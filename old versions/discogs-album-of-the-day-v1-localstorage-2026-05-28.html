AI Features

Add an Anthropic API key to unlock Claude's Take, Fun Facts, and Tracklist Highlights on each pick. Get a key at console.anthropic.com.

WordPress Publishing

Publish picks to your WordPress site. Install the CrateDig plugin on your site, then generate an Application Password in your WordPress user profile.

Accidentally removed local picks? Pull them back from your WordPress site into CrateDig history.

Export & Archive

Export your history year-by-year as JSON files to keep a permanent local record. Load an exported file into the Archive view to browse it read-only.

Value Tracking

Export your collection value snapshots as a JSON backup. Import a CSV (date, low, median, high) to load historical data.

Danger Zone

Reset & Disconnect removes all settings, credentials, collection data, and history from this browser. This cannot be undone.
